calc_channel_nuis(nu)

Return the channel-dependent effective principal quantum numbers nui.

The channel dependent effective principal quantum numbers nui are defined via

\[E = I_i - \frac{Ry}{2 \nu_i^2} = I_{\text{ref}} - \frac{Ry}{nu^2}\]

calc_k_matrix(nu)

Return the K-matrix in the collision (outer) channel frame.

The K-matrix is defined as

\[K = tan(\pi \mu) = U tan(\pi \mu_{\alpha}) U^T\]

where U is the frame transformation matrix and mu_{alpha} are the eigen quantum defects. The transpose \(U^T = U^{-1}\) holds because U is real and orthogonal.

Parameters:

nu (float) – Effective principal quantum number with reference to the lowest ionization threshold.

Return type:

ndarray[tuple[Any, ...], dtype[Any]]

Returns:

K-matrix in the collision (outer) channel frame, K = tan(pi mu).

calc_m_matrix(nu)

Return the M-matrix in the collision (outer) channel frame.

The M-matrix is defined as

\[M = tan(β) + K = tan(\pi \nu) + tan(\pi \mu)\]
Parameters:

nu (float) – Effective principal quantum number with reference to the lowest ionization threshold.

Return type:

ndarray[tuple[Any, ...], dtype[Any]]

Returns:

M-matrix in the collision (outer) channel frame, M = tan(β) + K.

calc_scaled_m_matrix(nu)

Return the scaled M-matrix in the collision (outer) channel frame.

The scaled M-matrix is defined as

\[M_{\text{scaled}} = \cos(\pi \nu) M = \sin(\pi \nu) + \cos(\pi \nu) K\]

We use this to improve numerical stability when finding roots of det(M) = 0. This is especially important for states with nu close to half integer.

Return type:

ndarray[tuple[Any, ...], dtype[Any]]

Parameters:

nu (float)
